# 坦克大战 Web 项目

这是一个通过大模型协作推进开发的实验项目，目标是做一款 `Web 版 2D 联机回合制坦克对战游戏`。

## 项目目标

游戏的核心体验是：

- 大地图上的 2 人联机回合制对战
- 左右两侧分布均衡、上下错落的短平台道路
- 炮弹抛物线、风力、仰角和蓄力带来的射击操作性
- 道路作为资源，可被逐步破坏、压塌和消耗
- 小地图、镜头拖动和距离感辅助玩家进行预判

这个项目当前优先采用“`先做可扩展骨架，再做战斗原型，再接联机`”的开发方式。

## 当前设计结论

### 战斗规则

- 左右方向键移动
- `Shift` 跳跃
- `Space` 按住蓄力，松开发射
- `Up / Down` 在蓄力期间调整仰角
- 回合制，每回合默认 10 秒
- 每回合允许移动并完成一次射击

### 地图规则

- 地图整体较大，不限制在单屏内
- 左右双方平台不直接接壤
- 每边都有相对均衡的多层短平台
- 底部有最终承接道路，避免无限坠落
- 上打下容易，下打上难
- 摧毁对方上层与中层道路后，对方会逐步跌落到底层

### 地形破坏规则

- 使用“像素表现 + 网格判定”的混合方案
- 视觉上保留像素级爆炸和坑洞
- 逻辑上通过 5 级道路强度控制脆弱、坍塌和空洞
- 危险平台在被坦克碾压或再次受击时会彻底损坏

## 当前工程结构

```text
apps/
  client/   React + Vite 客户端原型
  cocos-client/ Cocos Creator 客户端迁移落点
  server/   Node.js + WebSocket 服务端骨架
packages/
  shared/   地图、数值、规则、类型定义
docs/       设计文档、架构说明、界面示意图
```

## 本地开发

当前仓库已经具备工程结构，但还没有安装依赖。

### 安装

```bash
npm install
```

### 启动客户端

```bash
npm run dev:client
```

### 启动服务端

```bash
npm run dev:server
```
